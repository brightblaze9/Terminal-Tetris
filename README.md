# Terminal-Tetris

Instructions to run:
--------------------

 - Fire up the terminal
 - cd to the directory where play_tetris.py is located
 - run `python3 play_tetris.py` (Yes, please make sure you're running with Python 3)
 - Follow on screen instructions to play the game
